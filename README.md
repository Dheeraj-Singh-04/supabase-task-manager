# Supabase Task Manager

A simple Task Manager built using HTML, CSS, JavaScript, and Supabase.

## Features

- User Authentication
- Sign Up
- Login
- Logout
- Add Task
- Read Tasks
- Update Task Status
- Delete Task
- Row Level Security (RLS)
- Responsive UI

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Supabase
